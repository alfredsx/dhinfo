# Modul SX-INFO

Content-Modul zum erstellen individueller Seiten.

Es können statische Seiten erstellt, PHP-Code includiert,
externe sowie interne Seiten aufgerufen werden oder auch Dateien
wie z.B. PDF's eingebunden werden.

Das Modul ist frei klonbar, heisst es braucht nur der Verzeichnisname 
geändert werden um mehrere Instanzen zu haben.
